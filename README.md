# CleanWave.Gr

Public website for CleanWave.Gr - A Greek-based industrial laundry business specializing in clean technologies, advanced automation, and respect for people and the environment.

## About

CleanWave.Gr provides high-quality linen and towel services to hotels and restaurants, combining efficiency, care for the customer, cleanliness, and punctuality.

## Website Features

- **Clean, Modern Design** - Professional appearance with blue gradient theme
- **Greek Language** - All content in Greek (Ελληνικά)
- **Fully Responsive** - Works on desktop, tablet, and mobile devices
- **Accessible** - Keyboard navigation with visible focus states
- **SEO Optimized** - Proper meta tags and semantic HTML
- **Performance** - Minimal dependencies, optimized CSS

## Sections

1. **Hero** - Main landing section with call-to-action
2. **About** - Information about CleanWave.Gr
3. **Services** - Six service offerings (Hotels, Restaurants, Linens, Towels, Delivery, Quality)
4. **Values** - Six core values (Clean Technologies, Automation, Environment, People, Efficiency, Punctuality)
5. **Contact** - Contact information and details

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with responsive design
- `script.js` - JavaScript for smooth scrolling and animations

## Local Development

To run the website locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/index.html` in your browser.

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (no frameworks)
- Emoji icons for visual appeal
